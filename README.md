## What is changed in this fork?

"Too many tabs" error notification is customizable, badge can switch between different counting modes and it supports managed storage for rollouts via policy ([Example file](maxtablimit@cook_it.json)).

Unlike similar extensions it does not support per-window tab limits, only a global limit across all windows.

It is not very optimized as I lack experience outside of Excel-VBA, but it works and shouldn't slow down anything, at least it's not noticable on the old ~2009 dual-core Linux machines we use this on.

## What it does

This extension allows the user to specify an upper limit for tabs open at the same time.

## What it shows

It displays the number of currently opened tabs.
